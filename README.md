# Cybernetics Learning Lab

## Overview

Cybernetics Learning Lab is a long-term project documenting my journey from beginner Python programming to building intelligent adaptive systems.

The goal is not simply to learn software development, but to understand the principles that underlie intelligent behaviour, autonomous systems, cybernetics, control theory, robotics, artificial intelligence, and adaptive agents.

This repository serves as both a learning journal and a collection of increasingly sophisticated projects that build upon one another.

Each project introduces a new concept while reinforcing previous concepts, creating a structured path toward understanding and designing intelligent systems from first principles.

---

## Long-Term Vision

The ultimate objective is to understand how complex adaptive systems can:

* Sense their environment
* Maintain internal stability
* Make decisions
* Learn from experience
* Adapt to changing conditions
* Build internal models of the world
* Improve their behaviour over time

This repository explores these ideas through practical implementation rather than theory alone.

---

## Learning Philosophy

The projects in this repository follow several principles:

### Learn by Building

Concepts are learned through implementation rather than passive study.

### Incremental Complexity

Each project introduces only a small number of new ideas.

### Understand Every Line

Code generated by AI assistants must be understood, modified, and explained before being accepted.

### Simulation First

Complex systems are first explored through software simulation before considering hardware implementations.

### Systems Thinking

Projects focus on interactions between components rather than isolated pieces of code.

## Learning Roadmap

### Phase 1: Programming Foundations

#### 01 - Creature Simulator

Concepts:

* Variables
* Functions
* Loops
* Conditional logic
* Internal state

The first introduction to the idea that a system can possess and update state over time.

---

#### 02 - State Machine Robot

Concepts:

* Finite state machines
* Behaviour switching
* Decision making

Introduction to state-based control architectures used throughout engineering and robotics.

---

### Phase 2: Feedback and Control

#### 03 - Thermostat Controller

Concepts:

* Feedback loops
* Error signals
* Regulation

Introduction to cybernetic control systems.

---

#### 04 - Water Tank Controller

Concepts:

* Dynamic systems
* Disturbances
* Stability

Understanding how controllers interact with changing environments.

---

### Phase 3: Scientific Computing

#### 05 - PID Controller

Concepts:

* Proportional control
* Integral control
* Derivative control
* Control tuning

A foundation of modern industrial control systems.

---

### Phase 4: Adaptive Agents

#### 06 - Homeostatic Agent

Concepts:

* Internal needs
* Goal-directed behaviour
* Homeostasis

Building systems that actively maintain desired internal states.

---

#### 07 - World Model

Concepts:

* Environment representation
* Prediction
* Memory

Developing internal representations of the external world.

---

### Phase 5: Learning Systems

#### 08 - Adaptive Controller

Concepts:

* Self-adjustment
* Learning
* Adaptation

Controllers that improve their behaviour through experience.

---

### Phase 6: Integrated Architectures

#### 09 - EMO-0

Concepts:

* Sensing
* Control
* Memory
* Learning
* Internal state
* Adaptive behaviour

The integration of previous projects into a unified intelligent system simulation.

## Development Workflow

For every project:

1. Build the smallest working version.
2. Understand every line of code.
3. Explain the code in plain language.
4. Modify the system independently.
5. Document lessons learned.
6. Commit progress regularly.
